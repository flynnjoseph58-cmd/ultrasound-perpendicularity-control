# Ultrasound Perpendicularity Control

## Overview
Non-contact air-coupled ultrasonic perpendicularity control system 
mounted on a LoCoBot WidowX 250 6DOF arm. Uses echo amplitude and 
echo width feedback to autonomously maintain sensor orientation 
perpendicular to unknown surfaces without contact, coupling medium, 
or prior surface knowledge.

## Hardware
- LoCoBot with WidowX 250 6DOF arm
- HC-SR04 ultrasound sensor (early phases)
- 60kHz air-coupled ultrasonic transducer (later phases)

## Software
- Python, ROS, interbotix SDK

## Project Phases
- Phase 0: SDK familiarization
- Phase 1: Open loop straight line path following
- Phase 2: HC-SR04 height control
- Phase 3: Robustness testing
- Phase 4+: Acoustic sensing and perpendicularity control
